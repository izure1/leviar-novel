# 📖 Fumika Documentation Portal

**Fumika** 공식 문서 포털입니다.  
본 문서는 개발자가 코드를 복사하여 즉시 게임을 구동할 수 있도록 실용적인 가이드와 예제를 제공합니다.  

---

## 🏁 빠른 시작 안내 (Quick Navigation)

이 문서는 아래의 순서대로 읽는 것을 권장합니다.  
처음 오셨다면 튜토리얼부터 차근차근 따라 해보세요.  

*   **[📖 튜토리얼 (Tutorial)](./tutorial.md)**: 기초적인 장면(Scene) 작성부터 캐릭터 띄우기까지 단계별 코드를 제공합니다.  
*   **[📜 명령어 참조 (Commands)](./commands.md)**: 실제 연출에 사용되는 명령어들의 목록과 바로 쓸 수 있는 예제를 제공합니다.  
*   **[💡 핵심 개념 (Concepts)](./concepts.md)**: 변수를 다루거나 동적 분기를 작성할 때 필요한 필수 개념을 코드로 설명합니다.  
*   **[🪝 엔진 훅 (Hooks)](./modules/hooks/novel.md)**: 변수 변경, 씬 전환 등 엔진 내부 이벤트에 개입하는 방법을 안내합니다.  

---

## 🚀 문서 읽는 순서 추천

문서만 보고도 바로 복사해서 프로젝트에 적용할 수 있도록 구성되어 있습니다.  

1.  **[튜토리얼 (Tutorial)](./tutorial.md)**: 캐릭터를 띄우고 대사를 출력하는 코드를 작성해 봅니다.  
2.  **[명령어 참조 (Commands)](./commands.md)**: 화면 흔들림, 배경 전환 등 필요한 연출이 있다면 여기서 코드를 복사해서 사용합니다.  
3.  **[핵심 개념 (Concepts)](./concepts.md)**: 호감도에 따른 분기나 복잡한 조건 처리가 필요할 때 읽어봅니다.
