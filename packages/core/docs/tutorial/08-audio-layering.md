# 8. 오디오 레이어링: 배경음과 효과음

이 페이지에서는 씬에서 오디오를 재생하고 관리하는 방법을 배웁니다.  
화면에 출력되는 요소 외에도 배경음악과 효과음을 추가하여 당신의 상황을 생생하게 표현할 수 있습니다.

## 동기

텍스트와 이미지만 출력되는 씬은 단조로울 수 있습니다.  
특정 상황을 강조하거나 대화의 분위기를 전달할 때 소리가 없다면 당신의 의도를 명확하게 보여주기 어렵죠.

```typescript
// ❌ 이렇게 하면 안 됩니다
export default defineScene({ config })(() => [
  { type: 'character', name: 'fumika', image: 'idle:normal' },
  { 
    type: 'dialogue', 
    text: '좀 심심하지 않아?' 
  }
])
```

오디오 트랙을 적절히 분리하여 재생하면 상황을 훨씬 더 잘 전달할 수 있습니다.  
이제 오디오를 배경과 효과음으로 나누어 관리하는 방법을 살펴봅시다.

## 기본 사용법

[`audio`](../modules/audio.md) 명령어를 사용해 사운드를 재생할 수 있습니다.  
배경음악(`BGM`)과 효과음(`SE`)은 `name` 속성을 통해 서로 다른 트랙에 할당해 보세요.  
트랙이 다르면 당신은 두 소리를 동시에 겹쳐서 재생할 수 있습니다.

```typescript
// ✅ 이렇게 하세요
export default defineScene({ config })(() => [
  { 
    type: 'audio', 
    action: 'play', 
    name: 'bgm',      // 'bgm'이라는 이름의 오디오 슬롯(트랙)을 사용합니다
    src: 'bgm-room',  // 실제로 재생할 파일의 키값입니다
    repeat: true,     // 무한 반복 재생합니다
    volume: 0.4 
  },

  { 
    type: 'dialogue', 
    text: '역시 음악이 있으니까 훨씬 좋네요.' 
  },

  { 
    type: 'audio', 
    action: 'play', 
    name: 'se',       // 효과음은 'se' 슬롯을 사용해 배경음과 섞이지 않게 합니다
    src: 'se-heart' 
  },
  
  { 
    type: 'dialogue', 
    text: '호감도가 상승했습니다.' 
  }
])
```

동일한 `name` 트랙에 다른 오디오를 재생하면 기본적으로 이전 소리는 즉시 중단되고 새 소리가 재생됩니다.  
단, 이때 `duration` 속성을 지정하면 이전 소리는 서서히 작아지고 새 소리는 서서히 커지는 크로스페이드(Crossfade) 연출이 적용됩니다.  
배경음과 효과음을 동시에 겹쳐서 재생하고 싶다면 위 예시처럼 반드시 서로 다른 `name`을 지정해야 합니다.

> [!DANGER]
> `src` 속성에는 파일 경로를 직접 입력하면 안 됩니다.  
> 반드시 프로젝트 설정 파일의 `audios` 객체에 등록된 키값을 사용하세요.

> [!TIP]
> 배경음의 `volume`은 0.3에서 0.5 사이로 설정해 보세요.  
> 효과음이나 목소리가 묻히는 문제를 방지할 수 있습니다.

## 점진적 심화

오디오를 멈출 때 `stop` 액션을 사용합니다.  
이때 소리가 갑자기 끊기면 부자연스러울 수 있으므로 `duration` 속성을 사용해 서서히 볼륨을 줄이는 페이드아웃 효과를 줄 수 있습니다.

```typescript
// ✅ 페이드아웃 효과 적용
export default defineScene({ config })(() => [
  { 
    type: 'dialogue', 
    text: '이제 작별할 시간입니다.' 
  },

  // 'bgm' 트랙을 3초간 서서히 멈춥니다
  { 
    type: 'audio', 
    action: 'stop', 
    name: 'bgm', 
    duration: 3000 
  },

  { 
    type: 'dialogue', 
    text: '내일 또 놀러 오세요.' 
  }
])
```

지정된 시간(ms) 동안 볼륨이 0으로 떨어진 뒤에 트랙 재생이 완전히 종료됩니다.  
장면이 끝날 때 음악을 부드럽게 페이드아웃 처리해 보세요.

> [!WARNING]
> 너무 많은 수의 오디오 트랙을 동시에 재생하지 않도록 주의하세요.  
> 모바일 환경이나 구형 브라우저에서 성능 저하를 일으킬 수 있습니다.
