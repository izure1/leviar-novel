# 📈 이징 함수 (Easing Functions)

## 개요 (Overview)

이징 함수는 애니메이션의 진행 속도를 제어하여 움직임을 더 자연스럽게 만듭니다.  
수학적인 원리보다는 **화면에서 느껴지는 실제 움직임의 인상**을 기준으로 선택하세요.

---

## 1. 표준 움직임 (Standard)

가장 보편적으로 사용되는 부드러운 움직임들입니다.

| 명칭 | 느낌 (Feeling) | 추천 용도 |
| :--- | :--- | :--- |
| **`linear`** | 기계가 움직이듯 처음부터 끝까지 속도가 일정합니다. | 단순 루프, 등속도 이동 |
| **`easeInQuad`** | 처음엔 느긋하다가 끝날 때 살짝 속도가 붙습니다. | 화면 밖으로 퇴장하는 요소 |
| **`easeOutQuad`** | 처음엔 빠르다가 끝날 때 부드럽게 감속합니다. | 화면 안으로 등장하는 요소 |
| **`easeInOutQuad`** | 시작과 끝은 부드럽고 중간만 빠르게 이동합니다. | 캐릭터 이동, 카메라 팬 (기본값) |

---

## 2. 아주 부드러운 움직임 (Natural & Soft)

움직임의 변화가 거의 느껴지지 않을 정도로 매끄럽고 자연스럽습니다.

| 명칭 | 느낌 (Feeling) | 추천 용도 |
| :--- | :--- | :--- |
| **`easeInSine`** | 아주 조심스럽게 움직이기 시작합니다. | 은은한 안개 효과, 조명 변화 |
| **`easeOutSine`** | 깃털이 내려앉듯 아주 미세하게 속도가 줄어듭니다. | 텍스트 페이드인 |
| **`easeInOutSine`** | 마치 숨을 쉬는 것처럼 매우 자연스럽게 전환됩니다. | 대기 상태의 미세한 흔들림 |

---

## 3. 급격하고 절도 있는 움직임 (Sharp & Snappy)

속도 변화가 커서 눈에 확 띄며, 절도 있는 연출이 가능합니다.

| 명칭 | 느낌 (Feeling) | 추천 용도 |
| :--- | :--- | :--- |
| **`easeInCirc`** | 멈춰있는 듯하다가 갑자기 휙 하고 사라집니다. | UI 닫기, 빠른 퇴장 |
| **`easeOutCirc`** | 달려오던 물체가 벽에 부딪히듯 순식간에 감속합니다. | 버튼 클릭 반응, 메뉴 열기 |
| **`easeInOutCirc`** | 정지 상태에서 최고 속도까지 아주 짧게 전환됩니다. | 강렬한 화면 전환 |

---

## 4. 물리적인 반동과 탄성 (Dynamic)

물체가 가진 무게감이나 반동을 표현하여 생동감을 줍니다.  
※ `element` 등 일부 모듈에서는 위치나 크기 변화가 수반될 때 효과가 뚜렷합니다.

| 명칭 | 느낌 (Feeling) | 추천 용도 |
| :--- | :--- | :--- |
| **`easeOutBack`** | 목적지보다 살짝 더 갔다가 돌아오며 멈춥니다. | 카메라 줌, 위치 이동 강조 |
| **`easeInBack`** | 출발 전 뒤로 몸을 살짝 뺐다가 앞으로 나아갑니다. | 돌진 전 역동적인 움직임 |
| **`easeOutBounce`** | 바닥에 튀는 공처럼 통통 튀며 멈춥니다. | 낙하하는 물체 연출 |

---

## 사용 예시

```typescript
// 카메라가 목적지로 이동할 때 살짝 반동을 주며 도착
{ 
  type: 'camera-pan', 
  x: 200, 
  duration: 1000, 
  ease: 'easeOutBack' 
}

// 오버레이 텍스트가 사인 곡선을 그리며 아주 부드럽게 나타남
{ 
  type: 'overlay-text', 
  action: 'show', 
  name: 'title', 
  text: '제 1장', 
  duration: 2000, 
  ease: 'easeInOutSine' 
}

// UI 요소가 화면 안으로 부드럽게 페이드인
{ 
  type: 'element', 
  action: 'show', 
  id: 'panel', 
  duration: 400, 
  ease: 'easeOutQuad' 
}
```
