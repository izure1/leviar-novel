# 💬 Dialogue Module

---

## 1. 개요 (Overview)

`dialogue` 모듈은 비주얼 노벨의 가장 기본이 되는 **대화 및 나레이션 출력**을 담당합니다. 캐릭터의 이름(화자)과 대사를 화면 하단 대화창에 표시하며, 텍스트 타이핑 효과와 스타일 커스터마이징을 지원합니다.

### 주요 특징
*   **타이핑 효과**: 텍스트가 한 글자씩 출력되는 연출을 지원하며, 출력 속도를 조절할 수 있습니다.
*   **다중 라인 지원**: 하나의 커맨드로 여러 줄의 대사를 순차적으로 출력할 수 있습니다.
*   **자동 화자 해석**: `novel.config.ts`의 `characters`에 등록된 **캐릭터 키**를 `speaker`로 지정하면, 해당 캐릭터 정의에 설정된 **`name` 값이 화면에 이름으로 출력**됩니다. (등록되지 않은 이름을 직접 입력할 수도 있습니다.)
*   **텍스트 보간**: `{{var}}` 형식을 통해 변수 값을 실시간으로 대사에 반영합니다.

---

## 2. 핵심 예제 (Main Example)

### 기본 대화 및 나레이션

```ts
[
  // 1. 화자가 있는 대화
  { type: 'dialogue', speaker: 'heroine', text: '안녕! 오늘 날씨가 정말 좋다.' },

  // 2. 나레이션 (speaker 생략)
  { type: 'dialogue', text: '그녀는 밝게 웃으며 나에게 인사를 건넸다.' },

  // 3. 다중 라인 출력 (클릭 시마다 다음 줄로 진행)
  { 
    type: 'dialogue', 
    speaker: 'heroine', 
    text: [
      '있잖아, 사실 할 말이 있어...',
      '나랑 같이 축제에 가지 않을래?'
    ],
    speed: 50 // 타이핑 속도 지정 (ms)
  }
]
```

## 3. 커맨드 상세 (Command Reference)

### Dialogue 명령 (`DialogueCmd`)

| 속성 | 타입 | 필수 | 설명 |
| :--- | :--- | :---: | :--- |
| `type` | `'dialogue'` | ✅ | 대화 커맨드임을 명시합니다. |
| `text` | `string \| string[]` | ✅ | 출력할 대사 내용입니다. 배열인 경우 클릭 시 순차적으로 출력됩니다. |
| `speaker` | `string` | - | 화자의 키입니다. `characters`에 등록된 키를 사용하면 해당 캐릭터의 **`name`**이 표시됩니다. 등록되지 않은 이름을 직접 쓸 수도 있습니다. |
| `speed` | `number` | - | 타이핑 속도(ms)입니다. 값이 작을수록 빠릅니다. (0인 경우 즉시 출력) |

## 6. 주의 사항 (Edge Cases)

*   **입력 대기**: 대사 타이핑 중에 클릭하면 타이핑이 즉시 완료되며, 완료된 상태에서 다시 클릭해야 다음 대사나 커맨드로 넘어갑니다.
*   **화자 미지정**: `speaker` 속성을 생략하면 이름 표시창이 숨겨지고 나레이션 모드로 작동합니다.
*   **보간 우선순위**: 텍스트 내의 `{{ }}` 보간은 훅(`text-render`)이 실행되기 전에 먼저 처리됩니다.
