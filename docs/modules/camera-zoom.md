# 🔍 Camera Zoom Module

---

## 1. 개요 (Overview)

`camera-zoom` 커맨드는 카메라의 시야각(FOV) 또는 거리를 조절하여 화면을 확대하거나 축소합니다. 특정 대상을 강조하는 클로즈업 연출이나 전체 상황을 보여주는 와이드 샷 전환에 필수적인 기능을 제공합니다.

---

## 2. 핵심 예제 (Main Example)

### 클로즈업 연출
화면을 1.5배 확대하여 대상을 강조합니다.

```ts
{ 
  type: 'camera-zoom', 
  preset: 'close-up', 
  duration: 800 
}
```

### 시점 초기화
기본 배율(1.0)로 신속하게 복귀합니다.

```ts
{ 
  type: 'camera-zoom', 
  preset: 'reset', 
  duration: 400 
}
```

---

## 3. 커맨드 상세 (Command Reference)

### Camera Zoom 명령 (`CameraZoomCmd`)

| 속성 | 타입 | 필수 | 기본값 | 설명 |
| :--- | :--- | :---: | :---: | :--- |
| `type` | `'camera-zoom'` | ✅ | - | 커맨드 타입을 명시합니다. |
| `preset` | `ZoomPreset` | ✅ | - | 줌 배율 프리셋을 선택합니다. (하단 표 참조) |
| `duration` | `number` | - | 프리셋 기본값 | 애니메이션 지속 시간(ms)입니다. |

#### 줌 프리셋 (`ZoomPreset`)

| 프리셋 | 배율 (Scale) | 기본 지속 시간 | 설명 |
| :--- | :---: | :---: | :--- |
| `close-up` | `1.5` | `800ms` | 대상을 크게 강조합니다. |
| `medium` | `1.2` | `600ms` | 적당한 거리감을 유지하며 강조합니다. |
| `wide` | `0.92` | `800ms` | 평소보다 더 넓은 시야를 확보합니다. |
| `reset` | `1.0` | `600ms` | 기본 시점으로 돌아갑니다. |
| `inherit` | - | - | 이전 줌 상태를 유지합니다. |

---

## 4. 주의 사항 (Edge Cases)

* **상태 보존**: 줌 상태는 명시적으로 변경하거나 초기화하기 전까지 유지되며, 씬이 전환되어도 상위 씬의 `preserve` 설정에 따라 유지될 수 있습니다.
* **복합 연출**: `camera-pan`과 동시에 사용하여 특정 대상을 추적하며 확대하는 연출이 가능합니다.
