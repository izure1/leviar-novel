# 🖼️ Overlay Module

본 문서는 `overlay` 모듈을 사용하여 화면 위에 독립적인 시각 요소를 배치하고 연출하는 방법을 기술합니다.  

---

## 1. 개요 (Overview)

`overlay` 모듈은 게임 화면 위에 독립적인 텍스트나 이미지를 계층적으로 배치하여 연출의 밀도를 높이는 기능을 담당합니다.  
장 별 제목, 장소 안내, 캐릭터의 내면적 독백(Whisper), 또는 특정 위치에 고정되는 상징적 로고 등을 표시할 때 폭넓게 활용됩니다.  

이 모듈은 연출 대상과 방식에 따라 세 가지 세부 커맨드로 구성되어 있습니다.  
1.  **`overlay-text`**: 다양한 스타일의 텍스트 오버레이를 생성하고 제어합니다.  
2.  **`overlay-image`**: 화면 상에 독립적인 이미지 객체를 배치하고 관리합니다.  
3.  **`overlay-effect`**: 활성화된 오버레이 객체에 흔들림이나 맥동 등의 물리적 효과를 부여합니다.  

---

## 2. 핵심 예제 (Main Example)

### 타이틀 및 브랜드 로고 연출 구현
장면의 시작 부분에서 제목을 알리고 로고를 노출하는 일반적인 연출 흐름입니다.  

```ts
[
  // 1. 텍스트 오버레이 (사전 정의된 title 프리셋을 활용합니다)
  { 
    type: 'overlay-text', 
    action: 'show', 
    name: 'chapter_title', 
    text: '제 1장: 고요한 폭풍의 시작', 
    preset: 'title' 
  },

  // 2. 이미지 오버레이 (화면 중앙 상단 영역에 고정 배치합니다)
  { 
    type: 'overlay-image', 
    action: 'show', 
    name: 'main_logo', 
    src: 'logo_img', 
    x: 0.5, y: 0.3, 
    width: 400 
  },

  // 3. 배치된 로고 객체에 가벼운 물리 효과를 적용하여 주목도를 높입니다.  
  { 
    type: 'overlay-effect', 
    name: 'main_logo', 
    preset: 'shake', 
    duration: 500 
  },

  // 4. 연출 완료 후 특정 오버레이를 화면에서 서서히 소거합니다.  
  { type: 'overlay-text', action: 'hide', name: 'chapter_title', duration: 1000 }
]
```

## 3. 커맨드 상세 (Command Reference)

### Overlay Text (`overlay-text`)
| 속성 | 타입 | 필수 | 설명 |
| :--- | :--- | :---: | :--- |
| `action` | `'show' \| 'hide'` | ✅ | 오버레이의 가시성 상태를 결정합니다. |
| `name` | `string` | ✅ | 오버레이를 식별하기 위한 고유 이름입니다. |
| `text` | `string` | - | 표시할 텍스트 내용입니다. |
| `preset` | `string` | - | 스타일 프리셋을 지정합니다.  (`caption`, `title`, `whisper`) |
| `duration` | `number` | - | 나타나거나 사라지는 전환 애니메이션 시간(ms)입니다. |

### Overlay Image (`overlay-image`)
| 속성 | 타입 | 필수 | 설명 |
| :--- | :--- | :---: | :--- |
| `action` | `'show' \| 'hide'` | ✅ | 이미지 오버레이의 가시성을 결정합니다. |
| `name` | `string` | ✅ | 이미지를 식별하기 위한 고유 이름입니다. |
| `src` | `string` | - | 표시할 이미지의 에셋 식별자입니다. |
| `x`, `y` | `number` | - | 화면 내 좌표 비율입니다.  (0.0 ~ 1.0, 0.5가 중앙입니다) |
| `width`, `height` | `number` | - | 출력될 이미지의 크기(px)입니다. |

---

## 4. 텍스트 프리셋 안내 (Text Presets)

| 프리셋 이름 | 연출 목적 | 시각적 특징 |
| :--- | :--- | :--- |
| `title` | 챕터 제목, 주요 전환점 | 화면 중앙, 강조된 서체 및 큰 크기 |
| `caption` | 장소 및 시간 정보 전달 | 화면 하단, 표준적인 가독성 위주의 스타일 |
| `whisper` | 속삭임, 심리적 독백 | 화면 하단, 작은 서체와 낮은 투명도 |

---

## 5. 주의 사항 (Edge Cases)

*   **동일 식별자 갱신 정책**: 현재 화면에 표시 중인 오버레이와 동일한 `name`으로 새로운 `show` 명령을 하달하면, 기존 콘텐츠가 새로운 설정으로 부드럽게 전환되어 갱신됩니다.  
*   **레이어 계층 구조**: 오버레이 요소들은 일반적인 인물이나 배경 레이어보다 항상 상단에 위치하도록 설계되어 있습니다.  프리셋 설정에 따라 `whisper` < `caption` < `title` 순의 내부적 우선순위를 가집니다.  
*   **효과 연동 조건**: `overlay-effect` 명령은 대상이 되는 오버레이 객체가 화면에 실존하는 상태에서만 유효하게 작동합니다.  
*   **데이터 영속성**: 활성화된 모든 오버레이 상태는 엔진의 상태 시스템에 보존되어, 세이브 데이터 로드 시 동일한 지점에서 정확히 복원됩니다.
