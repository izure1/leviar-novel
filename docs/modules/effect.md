# ✨ Effect Module

---

## 1. 개요 (Overview)

`effect` 모듈은 게임 화면에 날씨나 분위기를 연출하기 위한 파티클 효과(비, 눈, 낙엽, 불꽃 등)를 추가하고 관리하는 기능을 제공합니다. 미리 정의된 물리 법칙과 시각적 설정을 바탕으로 고성능 파티클 시스템을 구동하며, 효과의 밀도나 이미지를 자유롭게 변경할 수 있습니다.

### 주요 특징
*   **다양한 프리셋**: 비, 눈, 벚꽃, 안개, 반딧불이 등 8종의 기본 효과를 지원합니다.
*   **커스텀 에셋**: 각 파티클에 사용될 이미지를 직접 지정할 수 있습니다.
*   **상태 유지**: 씬이 전환되어도 제거하기 전까지 효과가 계속 유지됩니다. (Save/Load 지원)

---

## 2. 핵심 예제 (Main Example)

### 효과 추가 및 제거

```ts
[
  // 1. 비 내리는 효과 추가 (전용 에셋 'rain_drop' 사용)
  { 
    type: 'effect', 
    action: 'add', 
    effect: 'rain', 
    src: 'rain_drop',
    rate: 150 // 파티클 밀도 조절
  },

  { type: 'dialogue', text: '갑자기 비가 내리기 시작했다.' },

  // 2. 비 효과 제거 (1.5초 동안 서서히 페이드 아웃)
  { 
    type: 'effect', 
    action: 'remove', 
    effect: 'rain', 
    duration: 1500 
  }
]
```

## 3. 커맨드 상세 (Command Reference)

### Effect 명령 (`EffectCmd`)

#### Action: `add` (효과 추가)
| 속성 | 타입 | 필수 | 설명 |
| :--- | :--- | :---: | :--- |
| `action` | `'add'` | ✅ | 효과를 추가함을 명시합니다. |
| `effect` | `EffectType` | ✅ | 추가할 효과의 종류입니다. (하단 목록 참조) |
| `src` | `string` | ✅ | 파티클 하나하나에 사용될 에셋 키입니다. |
| `rate` | `number` | - | 초당 파티클 생성 빈도입니다. 값이 클수록 효과가 진해집니다. (기본값은 효과마다 상이) |

#### Action: `remove` (효과 제거)
| 속성 | 타입 | 필수 | 설명 |
| :--- | :--- | :---: | :--- |
| `action` | `'remove'` | ✅ | 효과를 제거함을 명시합니다. |
| `effect` | `EffectType` | ✅ | 제거할 효과의 종류입니다. |
| `duration` | `number` | - | 효과가 완전히 사라지기까지의 페이드 아웃 시간(ms)입니다. (기본값: 600ms) |

## 4. 지원하는 효과 종류 (Effect Types)

| 종류 | 물리적 특징 (Movement Pattern) | 주요 연출 |
| :--- | :--- | :--- |
| `dust` | 아주 낮은 중력으로 느리게 부유하며, 불규칙하게 명멸합니다. | 햇살이 비치는 실내, 오래된 도서관 |
| `rain` | 높은 중력으로 빠르게 수직 낙하합니다. 생성 빈도가 매우 높습니다. | 비 내리는 실외, 소나기 |
| `snow` | 낮은 중력으로 천천히 떨어지며, 약간의 회전이 가해집니다. | 겨울, 추운 날씨, 고요한 분위기 |
| `sakura` | 불규칙한 회전과 함께 흩날리듯 천천히 떨어집니다. (Fluttering) | 봄, 꽃잎이 날리는 연출 |
| `sparkle` | 짧은 생존 시간 동안 반짝이다 사라지며 중력의 영향을 약간 받습니다. | 마법 연출, 환상적인 분위기 |
| `fog` | 매우 크고 반투명한 입자가 화면을 천천히 가로지르며 부유합니다. | 안개 낀 숲, 신비로운 장소 |
| `leaves` | 공기 저항과 강한 회전이 적용되어 불규칙하게 소용돌이치며 떨어집니다. | 가을, 바람 부는 숲, 낙엽 |
| `fireflies` | **역중력**이 적용되어 위로 천천히 떠오르며 좌우로 일렁입니다. | 밤의 숲, 몽환적인 연출 |

> [!TIP]
> **움직임과 이미지의 분리**: `effect` 속성은 **파티클이 어떻게 움직이는지(물리 법칙)**를 결정하고, `src` 속성은 **파티클이 어떻게 보이는지(이미지)**를 결정합니다. 예를 들어, `effect: 'snow'`에 불꽃 이미지를 `src`로 지정하면, 불꽃이 마치 눈송이처럼 천천히 회전하며 떨어지는 독특한 연출이 가능합니다.

---

## 5. 주의 사항 (Edge Cases)

*   **중첩 사용**: 동일한 `effect` 타입을 여러 번 `add` 하는 경우, 기존 효과를 대체합니다. 서로 다른 타입의 효과는 무제한으로 중첩이 가능합니다. (예: `rain`과 `fog` 동시 실행 가능)
*   **에셋 준비**: 각 효과에 맞는 적절한 투명 배경 이미지를 `assets`에 먼저 등록해야 합니다.
*   **성능 고려**: `rate` 값을 과도하게 높이면 저사양 기기에서 성능 저하가 발생할 수 있으므로 적절한 값을 테스트하며 설정하세요.
