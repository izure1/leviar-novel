# ✨ 파티클 효과 (Effect)

## 개요 (Overview)

`effect` 모듈은 비, 눈, 벚꽃 같은 날씨나 반딧불이 같은 시각 파티클 이펙트를 화면에 추가하고 지우는 기능입니다.  

## 옵션 상세 (Properties)

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :---: | :--- |
| **`type`** | `'effect'` | 필수 | 커맨드 타입 |
| **`action`** | `'add' \| 'remove'` | 필수 | 효과 활성화(`add`) 또는 비활성화(`remove`) 선택 |
| **`effect`** | `string` | 필수 | 사용할 물리 패턴 이름 (`rain`, `snow`, `sakura`, `fireflies`, `fog` 등) |
| **`src`** | `string` | 필수 (`add`) | 파티클 개체 이미지로 사용할 에셋 식별자 키 |
| **`rate`** | `number` | (기본값) | 초당 파티클 생성 빈도(밀도). 수치가 클수록 많이 나옵니다. |
| **`duration`** | `number` | (기본값) | `remove`를 호출했을 때, 파티클 개수가 서서히 줄어들며 잦아드는 시간(ms) |

### 이펙트 패턴 목록 (Presets)

명령어의 `effect` 속성에 사용할 수 있는 8가지 물리 파티클 이펙트 패턴들입니다.  

| 이름 | 권장 렌더링(src) 에셋 | 상세 연출 방향 |
| :--- | :--- | :--- |
| **`rain`** | 빗방울, 물방울 이미지 | 위에서 아래로 빠른 속도로 떨어지며 화면을 가로지릅니다. |
| **`snow`** | 눈송이, 하얀 원 이미지 | 위에서 아래로 천천히 떨어지며 좌우로 살짝 흔들립니다. |
| **`sakura`** | 벚꽃잎, 꽃잎 이미지 | 대각선 아래로 흩날리며 궤적이 유려하게 흔들립니다. |
| **`fireflies`** | 빛나는 구, 별 이미지 | 화면 아래에서 위로 느리게 솟아오르며 반짝입니다. |
| **`fog`** | 구름 텍스처, 안개 이미지 | 화면 전체를 좌에서 우로 매우 느리게 흐르며 덮습니다. |
| **`dust`** | 먼지, 작은 점 텍스처 | 공기 중에 무중력 상태처럼 부유하는 먼지 입자를 연출합니다. |
| **`sparkle`** | 반짝이는 별, 마법 효과 | 몽환적이고 마법적인 빛무리가 반짝거리며 생성됩니다. |
| **`leaves`** | 나뭇잎 텍스처 | 바람에 흩날리듯 이리저리 떨어지는 나뭇잎을 연출합니다. |


## 핵심 예제 (Main Example)

### 날씨 효과 활성화 및 부드러운 비활성화

```typescript
import { defineScene } from 'fumika'
import config from './novel.config'

export default defineScene({ config })(() => [
  // 1. 비 내리는 효과를 화면에 켭니다
  { 
    type: 'effect', 
    action: 'add', 
    effect: 'rain', 
    src: 'rain-drop', // config.assets에 등록된 빗방울 이미지 키
    rate: 150 
  },

  { type: 'dialogue', text: '거센 빗줄기가 쏟아지기 시작했다.' },

  // 2. 비 효과를 1.5초(1500ms) 동안 서서히 줄이면서 소거합니다
  { 
    type: 'effect', 
    action: 'remove', 
    effect: 'rain', 
    duration: 1500 
  }
])
```

## 주의 사항 (Edge Cases)

| 상황 | 설명 및 해결 방법 |
| :--- | :--- |
| **물리와 시각의 결합** | `effect: 'snow'`에 불꽃놀이 이미지를 `src`로 주면, 불꽃놀이 파티클이 눈처럼 빙글빙글 돌면서 떨어지는 몽환적인 연출을 만들 수 있습니다. |
| **서로 다른 날씨 섞어 쓰기** | 눈(`snow`)과 안개(`fog`)처럼 패턴 이름이 다르면 서로 덮어씌워지지 않고 화면에 동시에 여러 효과가 공존할 수 있습니다. |
| **상태 영구 보존** | `action: 'remove'`로 명시적으로 지우기 전까지는 씬이 전환되어도 화면에 파티클이 계속 그려집니다. |
