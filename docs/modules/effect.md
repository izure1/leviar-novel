# ✨ Effect Module

본 문서는 `effect` 모듈을 사용하여 화면에 날씨나 분위기를 연출하기 위한 파티클 효과를 추가하고 관리하는 방법을 기술합니다.  

---

## 1. 개요 (Overview)

`effect` 모듈은 비, 눈, 낙엽, 불꽃 등 화면 전반에 걸쳐 역동적인 시각 요소를 부여하는 파티클 시스템을 담당합니다.  
미리 정의된 물리 법칙과 정교한 시각적 설정을 바탕으로 고성능 파티클을 생성하며, 효과의 밀도나 사용되는 이미지를 자유롭게 변경하여 풍부한 몰입감을 제공합니다.  

### 주요 특징
*   **다양한 물리 프리셋**: 비, 눈, 벚꽃, 안개, 반딧불이 등 8종 이상의 검증된 움직임 패턴을 지원합니다.  
*   **커스텀 에셋 매핑**: 각 파티클에 사용될 이미지를 직접 지정할 수 있어, 독창적인 시각 연출이 가능합니다.  
*   **상태 영속성 보장**: 씬이 전환되어도 명시적으로 제거하기 전까지 효과가 지속되며, 세이브 데이터 로드 시 모든 파티클 상태가 정확히 복원됩니다.  

---

## 2. 핵심 예제 (Main Example)

### 기상 효과 추가 및 자연스러운 소거 구현

```ts
[
  // 1. 비 내리는 효과를 추가합니다.  (전용 에셋 'rain_drop'을 활용합니다)
  { 
    type: 'effect', 
    action: 'add', 
    effect: 'rain', 
    src: 'rain_drop',
    rate: 150 // 초당 파티클 생성 밀도를 설정합니다.  
  },

  { type: 'dialogue', text: '갑자기 하늘이 어두워지더니, 거센 빗줄기가 쏟아지기 시작했습니다.' } ,

  // 2. 비 효과를 제거합니다.  (1.5초 동안 서서히 개체수를 줄이며 소거합니다)
  { 
    type: 'effect', 
    action: 'remove', 
    effect: 'rain', 
    duration: 1500 
  }
]
```

## 3. 커맨드 상세 (Command Reference)

### Effect 명령 (`EffectCmd`)

#### Action: `add` (효과 활성화)
| 속성 | 타입 | 필수 | 설명 |
| :--- | :--- | :---: | :--- |
| `action` | `'add'` | ✅ | 효과를 화면에 추가함을 명시합니다. |
| `effect` | `string` | ✅ | 추가할 파티클의 물리 이동 패턴(EffectType)입니다. |
| `src` | `string` | ✅ | 각 파티클 개체에 사용될 에셋의 키 이름입니다. |
| `rate` | `number` | - | 파티클의 생성 빈도를 조절합니다.  값이 클수록 연출이 농후해집니다. |

#### Action: `remove` (효과 비활성화)
| 속성 | 타입 | 필수 | 설명 |
| :--- | :--- | :---: | :--- |
| `action` | `'remove'` | ✅ | 현재 실행 중인 효과를 제거함을 명시합니다. |
| `effect` | `string` | ✅ | 제거하고자 하는 효과의 종류입니다. |
| `duration` | `number` | - | 효과가 완전히 사라지기까지의 페이드 아웃 시간(ms)입니다. |

---

## 4. 지원하는 물리 패턴 안내 (Effect Types)

| 패턴 키 | 물리적 특성 (Movement Pattern) | 권장 연출 |
| :--- | :--- | :--- |
| `dust` | 초저중력 부유 및 불규칙한 명멸 | 햇살이 드는 실내, 정적인 공간 |
| `rain` | 높은 중력을 통한 수직 낙하 및 고밀도 생성 | 폭우, 소나기 등 기상 상황 |
| `snow` | 낮은 중력에서의 완만한 낙하와 미세 회전 | 추운 겨울, 고요하고 평화로운 분위기 |
| `sakura` | 불규칙한 회전과 공기 저항이 적용된 흩날림 | 봄의 정취, 꽃잎이 날리는 서정적 연출 |
| `sparkle` | 짧은 생존 주기와 미세한 중력 반응 | 마법 효과, 환상적인 시각 연출 |
| `fog` | 대형 파티클의 완만한 수평 부유 및 확산 | 신비로운 숲, 안개 낀 고딕풍 장소 |
| `leaves` | 강한 회전력과 소용돌이 형태의 낙하 궤적 | 바람 부는 가을 산책로, 낙엽 연출 |
| `fireflies` | **역중력** 적용을 통한 상승 및 수평 일렁임 | 밤의 숲, 비현실적이고 몽환적인 연출 |

---

## 5. 주의 사항 (Edge Cases)

*   **중첩 및 대체 정책**: 동일한 `effect` 타입을 중복하여 `add` 할 경우 기존 설정이 최신 정보로 대체됩니다.  서로 다른 물리 패턴(예: `rain`과 `fog`)은 제한 없이 동시 호출이 가능합니다.  
*   **물리와 시각의 결합**: `effect` 속성은 **움직임의 법칙**을, `src` 속성은 **시각적 형태**를 결정합니다.  예를 들어 `effect: 'snow'`에 불꽃 이미지를 지정하면, 불꽃이 눈송이처럼 완만하게 회전하며 떨어지는 독특한 분위기를 자아낼 수 있습니다.  
*   **성능 최적화 권장**: `rate` 값을 과도하게 상향 조정하면 하드웨어 사양에 따라 프레임 저하가 발생할 수 있으므로, 연출 의도와 기기 성능 사이의 적절한 균형점을 테스트를 통해 확정하시기 바랍니다.  
*   **영속적 상태 보존**: 현재 활성화된 파티클의 모든 물리 상태는 엔진에 의해 기록되어 세이브 데이터에 안전하게 포함됩니다.
