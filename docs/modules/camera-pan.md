# 🎞️ Camera Pan Module

본 문서는 `camera-pan` 커맨드를 사용하여 카메라의 시점을 X, Y축으로 이동시키고 시선의 흐름을 제어하는 방법을 기술합니다.  

---

## 1. 개요 (Overview)

`camera-pan` 커맨드는 카메라의 렌즈가 바라보는 지점을 정교하게 이동시키는 기능을 담당합니다.  
캐릭터의 동선을 따라가거나, 배경의 숨겨진 정보를 노출하는 등 시나리오의 흐름에 따른 시각적 유도를 구현할 때 필수적으로 사용됩니다.  

---

## 2. 핵심 예제 (Main Example)

### 프리셋을 활용한 시점 이동
사전에 정의된 위치로 카메라 시점을 신속하게 이동시킵니다.  

```ts
{ 
  type: 'camera-pan', 
  position: 'left', 
  duration: 1000 
}
```

### 정교한 좌표 기반 이동
기준 위치에서 상세한 픽셀 단위 오프셋을 적용하여 시점을 조절합니다.  

```ts
{ 
  type: 'camera-pan', 
  position: 'center', 
  x: 300, 
  y: 100, 
  duration: 1500 
}
```

---

## 3. 커맨드 상세 (Command Reference)

### Camera Pan 명령 (`CameraPanCmd`)

| 속성 | 타입 | 필수 | 기본값 | 설명 |
| :--- | :--- | :---: | :---: | :--- |
| `type` | `'camera-pan'` | ✅ | - | 커맨드 타입을 명시합니다. |
| `position` | `string` | ✅ | - | 기준이 될 위치 프리셋을 선택합니다. |
| `x` | `number` | - | `0` | 선택된 기준 위치에서의 추가 X 좌표 오프셋(px)입니다. |
| `y` | `number` | - | `0` | 선택된 기준 위치에서의 추가 Y 좌표 오프셋(px)입니다. |
| `duration` | `number` | - | `1000` | 이동 애니메이션이 완료될 때까지의 시간(ms)입니다. |

#### 패닝 프리셋 명세 (`PanPreset`)

| 프리셋 이름 | 기준 좌표 (X, Y) | 연출 의도 |
| :--- | :---: | :--- |
| `center` | `(0, 0)` | 화면의 정중앙을 조준합니다. |
| `left` | `(-200, 0)` | 왼쪽 영역으로 시선을 옮깁니다. |
| `right` | `(200, 0)` | 오른쪽 영역으로 시선을 옮깁니다. |
| `up` | `(0, 200)` | 화면의 상단부를 비춥니다. |
| `down` | `(0, -200)` | 화면의 하단부를 비춥니다. |
| `"n/d"` | `(x, 0)` | 분수 형태의 비율 좌표를 지원합니다.  (예: `"1/2"`) |

---

## 4. 주의 사항 (Edge Cases)

*   **좌표 체계 안내**: `fumika` 엔진의 Y축은 위로 올라갈수록 값이 증가하는 좌표계를 채택하고 있습니다.  따라서 시점을 위로 올리려면 `y` 값을 양수로, 아래로 내리려면 음수로 지정해 주시기 바랍니다.  
*   **줌 상태와의 상호작용**: 화면이 크게 확대(`camera-zoom`)된 상태에서 패닝을 수행하면 시점 이동의 속도가 상대적으로 더욱 역동적으로 체감됩니다.  
*   **캐릭터 추적 권장 사항**: 특정 캐릭터의 위치를 정확히 조준해야 하는 경우에는 본 커맨드보다는 [캐릭터 포커스 (character-focus)](./character-focus.md) 커맨드를 활용하는 것이 더욱 효율적입니다.  
*   **애니메이션 중첩**: 이동 도중 새로운 패닝 명령이 전달되면, 이전 이동 경로는 무시되고 새로운 목표 지점을 향해 부드럽게 궤적을 수정합니다.
