# 🎞️ Camera Pan Module

---

## 1. 개요 (Overview)

`camera-pan` 커맨드는 카메라의 시점을 X, Y축으로 이동시킵니다. 캐릭터의 움직임을 따라가거나, 배경의 특정 부분을 비추는 등의 연출을 구현할 때 사용됩니다.

---

## 2. 핵심 예제 (Main Example)

### 프리셋 기반 이동
왼쪽으로 시점을 이동시킵니다.

```ts
{ 
  type: 'camera-pan', 
  position: 'left', 
  duration: 1000 
}
```

### 정교한 좌표 이동
정중앙에서 오른쪽으로 300px, 위로 100px 이동합니다.

```ts
{ 
  type: 'camera-pan', 
  position: 'center', 
  x: 300, 
  y: 100, 
  duration: 1500 
}
```

---

## 3. 커맨드 상세 (Command Reference)

### Camera Pan 명령 (`CameraPanCmd`)

| 속성 | 타입 | 필수 | 기본값 | 설명 |
| :--- | :--- | :---: | :---: | :--- |
| `type` | `'camera-pan'` | ✅ | - | 커맨드 타입을 명시합니다. |
| `position` | `PanPreset` | ✅ | - | 기준 위치 프리셋을 선택합니다. (하단 표 참조) |
| `x` | `number` | - | `0` | 기준 위치에서의 추가 X 좌표 오프셋(px)입니다. |
| `y` | `number` | - | `0` | 기준 위치에서의 추가 Y 좌표 오프셋(px)입니다. |
| `duration` | `number` | - | `1000` | 이동 애니메이션 지속 시간(ms)입니다. |

#### 패닝 프리셋 (`PanPreset`)

| 프리셋 | 좌표 (X, Y) | 설명 |
| :--- | :---: | :--- |
| `center` | `(0, 0)` | 화면 중앙을 조준합니다. |
| `left` | `(-200, 0)` | 왼쪽 방향을 비춥니다. |
| `right` | `(200, 0)` | 오른쪽 방향을 비춥니다. |
| `up` | `(0, 200)` | 위쪽 방향을 비춥니다. |
| `down` | `(0, -200)` | 아래쪽 방향을 비춥니다. |
| `"n/d"` | `(x, 0)` | 분수 형태의 비율 좌표를 지원합니다. (예: `"1/2"`) |

---

## 4. 주의 사항 (Edge Cases)

* **좌표 체계**: 엔진의 Y축은 위로 갈수록 값이 커지는 형태입니다. 따라서 시점을 위로 올리려면 `y` 값을 양수로, 아래로 내리려면 음수로 지정해야 합니다.
* **패닝과 줌의 상관관계**: 시점이 크게 확대(`camera-zoom`)된 상태에서 패닝을 수행하면 화면의 이동이 더욱 두드러져 보입니다.
* **캐릭터 포커스**: 특정 캐릭터를 조준하는 경우에는 본 커맨드 대신 `character-focus`를 사용하는 것이 훨씬 편리합니다.
