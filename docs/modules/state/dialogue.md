# 🎨 Dialogue State & Layout

본 문서는 `dialogue` 모듈의 시각적 구성 요소(배경 패널, 화자 식별자, 대사 텍스트)와 배치 레이아웃을 결정하는 `DialogueSchema`에 대해 기술합니다.  

---

## 1. 개요 (Overview)

대화창의 상태 설정은 비주얼 노벨의 가장 핵심적인 사용자 인터페이스인 대사 영역의 가독성과 분위기를 결정합니다.  
배경 패널의 질감부터 텍스트의 행간까지 정밀한 조정이 가능하며, 이를 통해 시나리오의 감동을 플레이어에게 온전히 전달할 수 있습니다.  

| 속성 그룹 명칭 | 상세 설명 |
| :--- | :--- |
| **`bg`** | 대화창 전체를 구성하는 배경 패널의 형태 및 색상 스타일입니다. |
| **`speaker`** | 캐릭터의 성명이 표시되는 텍스트의 서체 및 시각적 설정입니다. |
| **`text`** | 실제 대사가 출력되는 영역의 텍스트 레이아웃 및 스타일 설정입니다. |
| **`layout`** | 창 내부 구성 요소들 사이의 여백 및 수직적 배치를 정의합니다. |

---

## 2. 레이아웃 상세 명세 (`DialogueLayout`)

대화창 내부의 시각적 균형을 유지하기 위한 간격 설정입니다.  

| 속성 명칭 | 데이터 타입 | 기본값 | 설명 |
| :--- | :--- | :---: | :--- |
| **`panelPaddingLeft`** | `number` | `48` | 대화창 패널 좌측 끝에서 텍스트 시작점까지의 여백(px)입니다. |
| **`panelPaddingRight`** | `number` | `48` | 대화창 패널 우측 끝에서 텍스트가 끝나는 지점까지의 여백입니다. |
| **`panelPaddingTop`** | `number` | `24` | 패널 상단 경계에서 화자 이름이 나타나는 지점까지의 거리입니다. |
| **`panelPaddingBottom`** | `number` | `24` | 대화창 패널 하단 테두리에서 대사 텍스트 끝까지의 거리입니다. |
| **`speakerTextGap`** | `number` | `8` | 화자 이름 영역과 실제 대사 텍스트 영역 사이의 수직 간격입니다. |

---

## 3. 스타일링 활용 사례

### 감성적인 반투명 대화창 및 텍스트 레이아웃 설정
가독성을 확보하면서도 배경 연출을 가리지 않는 세련된 대화창 구성 예시입니다.  

```ts
export default defineScene({
  initial: {
    dialogue: {
      // 배경 패널 스타일 설정
      bg: { 
        height: 200, 
        color: 'rgba(20, 20, 40, 0.85)', // 깊이감 있는 남색 반투명 배경
        borderRadius: 16
      },
      // 대사 텍스트 상세 스타일
      text: { 
        fontSize: 22,
        lineHeight: 1.8, // 가독성을 위한 충분한 행간 확보
        color: '#f8f9fa'
      }
    }
  }
})
```

---

## 4. 상세 속성 가이드

### 배경 레이어 (`bg`)
대화창의 전체적인 무게감과 형태를 결정합니다.  
*   **`height`**: 대화창의 세로 크기(px)를 지정합니다.  별도 지정이 없을 경우 화면 전체 높이의 약 28%가 할당됩니다.  
*   기타 표준 스타일 속성(`color`, `opacity`, `border`, `shadow` 등)을 완벽하게 수용합니다.  

### 텍스트 오브젝트 (`speaker` & `text`)
가독성을 최우선으로 고려한 텍스트 스타일링을 지원합니다.  
*   **`fontSize`**, **`color`**, **`fontWeight`**, **`fontFamily`** 등을 통해 시인성을 조절할 수 있습니다.  
*   **`textShadow`** 속성을 적절히 활용하면 밝은 배경 위에서도 텍스트가 명확하게 인지되도록 보정할 수 있습니다.  

---

## 5. 주의 사항 (Edge Cases)

*   **레이어 우선순위(Z-Index)**: 대화창의 기본 `zIndex`는 환경 연출 요소보다 상위인 `400` 계층으로 설정되어 있습니다.  다른 커스텀 UI와 중첩될 경우 이를 정교하게 조정해 주시기 바랍니다.  
*   **해상도 독립성**: `panelPadding` 속성은 픽셀 단위로 엄격하게 적용되므로, 다양한 해상도 환경에서도 일관된 여백의 미를 유지합니다.  
*   **상태 영속성**: 런타임에 `ui` 커맨드를 통해 변경된 대화창 스타일은 세이브 데이터에 즉시 반영되어 로드 시 완벽하게 재현됩니다.
