# 🎭 Mood Module

---

## 1. 개요 (Overview)

`mood` 모듈은 화면 전체에 컬러 필터, 비네팅(Vignette), 조명 효과 등을 적용하여 게임의 시간대, 장소의 분위기, 또는 캐릭터의 심리 상태를 시각적으로 연출합니다. 단순한 색상 오버레이를 넘어 블렌드 모드(Multiply, Screen 등)와 실시간 조명 깜빡임(Flicker) 효과를 결합하여 풍부한 몰입감을 제공합니다.

### 주요 특징
*   **다양한 프리셋**: 낮, 밤, 노을, 세피아, 공포 등 15종 이상의 분위기 설정을 즉시 사용할 수 있습니다.
*   **블렌드 모드**: 각 무드에 최적화된 블렌드 모드가 자동으로 적용되어 배경 및 캐릭터와 자연스럽게 어우러집니다.
*   **조명 애니메이션**: 촛불처럼 일렁이거나 전등이 깜빡이는 등의 동적인 조명 효과를 지원합니다.
*   **다중 레이어**: 여러 개의 무드를 중첩하여 더 복잡한 분위기를 만들어낼 수 있습니다.

## 2. 핵심 예제 (Main Example)

### 시간대 변경 및 특수 효과

```ts
[
  // 1. 기본 무드 설정 (노을 분위기)
  { 
    type: 'mood', 
    mood: 'sunset', 
    intensity: 0.4, 
    duration: 2000 
  },

  { type: 'dialogue', text: '창밖으로 붉은 노을이 드리우기 시작했다.' },

  // 2. 무드 추가 및 깜빡임 효과 (공포 분위기 중첩)
  { 
    type: 'mood', 
    action: 'add', 
    mood: 'horror', 
    intensity: 0.6, 
    flicker: 'flicker' // 조명이 깜빡이는 효과
  },

  { type: 'dialogue', text: '갑자기 주변이 붉게 변하며 전등이 깜빡인다.' },

  // 3. 모든 무드 제거 및 초기화
  { type: 'mood', mood: 'none', duration: 1000 }
]
```

## 3. 커맨드 상세 (Command Reference)

### Mood 명령 (`MoodCmd`)

| 속성 | 타입 | 필수 | 설명 |
| :--- | :--- | :---: | :--- |
| `type` | `'mood'` | ✅ | 무드 커맨드임을 명시합니다. |
| `action` | `'add' \| 'remove'` | - | 수행할 동작입니다. 생략 시 기존 무드들을 모두 제거하고 현재 무드만 적용합니다. |
| `mood` | `MoodType` | ✅ | 적용할 무드의 종류입니다. (하단 목록 참조) |
| `intensity` | `number` | - | 무드 효과의 강도(0~1)입니다. 값이 클수록 진해집니다. |
| `flicker` | `FlickerPreset` | - | 조명이 일렁이거나 깜빡이는 효과를 적용합니다. (`candle`, `flicker`, `strobe`) |
| `duration` | `number` | - | 무드 전환 애니메이션의 지속 시간(ms)입니다. (기본값: 800ms) |
| `disable` | `boolean` | - | `true`인 경우, 전환 애니메이션이 진행되는 동안 사용자 입력을 차단합니다. |

## 4. 지원하는 무드 종류 (Mood Types)

| 종류 | 설명 | 주요 연출 |
| :--- | :--- | :--- |
| `day` | 밝은 낮 | 평상시, 화창한 야외 |
| `night` | 어두운 밤 | 밤 시간대, 어두운 실내 |
| `dawn` | 새벽 | 해 뜨기 전, 푸르스름한 분위기 |
| `sunset` | 노을 | 해 질 녘, 주황빛 분위기 |
| `foggy` | 안개 | 흐린 날씨, 신비로운 장소 |
| `sepia` | 세피아 | 과거 회상, 오래된 사진 느낌 |
| `cold` | 차가움 | 냉정한 분위기, 겨울, 긴장감 |
| `noir` | 느와르 | 흑백 영화 느낌, 고독한 연출 |
| `horror` | 공포 | 붉고 어두운 분위기, 위기 상황 |
| `flashback` | 회상 | 밝고 뽀얀 느낌의 회상 장면 |
| `dream` | 꿈 | 몽환적이고 보랏빛이 도는 연출 |
| `danger` | 위험 | 강한 붉은색, 경고 상황 |
| `spot` | 스포트라이트 | 중앙 집중 조명 |
| `warm` | 따뜻함 | 포근한 실내, 행복한 분위기 |

---

## 5. 주의 사항 (Edge Cases)

*   **덮어쓰기 vs 추가**: `action: 'add'`를 명시하지 않으면 기존에 적용되어 있던 모든 무드가 자동으로 제거됩니다. 단일 무드만 사용할 때는 생략하는 것이 관리하기 편합니다.
*   **none 무드**: `mood: 'none'`을 사용하면 현재 적용된 모든 무드 레이어를 제거하고 원래 화면 상태로 되돌립니다.
*   **성능**: 너무 많은 무드 레이어를 중첩(`action: 'add'`)하면 렌더링 성능에 영향을 줄 수 있으므로 적절히 조절하세요.
