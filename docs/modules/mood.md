# 🎭 무드 필터 (Mood)

## 개요 (Overview)

`mood` 모듈은 화면 전체에 컬러 필터를 씌우거나 블렌드 모드를 적용하여 게임의 시간대(낮, 밤)나 분위기를 연출합니다.  

## 옵션 상세 (Properties)

명령어 객체에 사용할 수 있는 모든 속성들의 목록입니다.  

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :---: | :--- |
| **`type`** | `'mood'` | 필수 | 커맨드 타입 |
| **`action`** | `'add' \| 'remove'` | (교체) | 무드 추가 혹은 제거. 생략하면 기존 무드를 다 지우고 덮어씁니다. |
| **`mood`** | `string` | 필수 | 적용할 프리셋 이름 (`sunset`, `night`, `horror` 등) |
| **`intensity`** | `number` | `1.0` | 효과의 강도 및 불투명도 (`0.0` ~ `1.0`) |
| **`flicker`** | `string` | (없음) | 조명 깜빡임 애니메이션 (아래 표 참고) |
| **`duration`** | `number` | 프리셋 기본값 | 분위기가 완전히 바뀔 때까지 전환에 걸리는 시간(ms) |

### 무드 프리셋 목록 (Mood Presets)

`mood` 속성에 문자로 입력할 수 있는 필터의 종류입니다. `none`을 넣으면 모든 무드가 즉시 해제됩니다.  

| 이름 | 분위기 및 연출 효과 |
| :--- | :--- |
| **`none`** | 무드 필터를 완전히 해제합니다. |
| **`day`** | 밝고 따뜻한 아침/낮 분위기의 부드러운 스크린 필터입니다. |
| **`night`** | 짙은 푸른빛과 강력한 비네팅이 결합된 밤 분위기입니다. |
| **`dawn`** | 서늘하고 푸르스름한 새벽빛을 연출합니다. |
| **`sunset`** | 붉은빛이 강하게 도는 저녁놀/석양 분위기입니다. |
| **`foggy`** | 채도를 낮추고 뿌옇게 흐려지는 안개 효과입니다. |
| **`sepia`** | 오래된 사진이나 과거 회상에 어울리는 세피아 톤입니다. |
| **`cold`** | 차갑고 창백한 푸른빛의 스산한 분위기입니다. |
| **`noir`** | 채도를 극단적으로 낮추고 명암을 강조하는 느와르 분위기입니다. |
| **`horror`** | 화면 주변이 어두워지고 피처럼 붉은 기운이 감도는 공포 연출입니다. |
| **`flashback`** | 하얗게 빛이 바래는 회상 또는 섬광 연출입니다. |
| **`dream`** | 보랏빛이 감도는 몽환적인 꿈속 연출입니다. |
| **`danger`** | 붉은빛과 비네팅을 결합하여 위험한 상황을 묘사합니다. |
| **`spot`** | 중앙만 밝고 주변이 어두운 핀 조명(스포트라이트) 연출입니다. |
| **`ambient`** | 공간 전체를 감싸는 부드럽고 따뜻한 조명 효과입니다. |
| **`warm`** | 노란빛이 강한 강렬하고 따뜻한 조명 효과입니다. |

### 조명 깜빡임 효과 (Flicker Presets)

`flicker` 속성에 추가하여 화재, 형광등 고장 등을 연출합니다.  

| 이름 | 효과 설명 |
| :--- | :--- |
| **`candle`** | 바람에 흔들리는 촛불처럼 밝기가 부드럽고 불규칙하게 일렁입니다. |
| **`flicker`** | 수명이 다 되어가는 형광등처럼 다소 거칠게 깜빡입니다. |
| **`strobe`** | 클럽 조명이나 번개처럼 극단적이고 매우 강하게 깜빡입니다. |

## 핵심 예제 (Main Example)

### 시간대 전환

```typescript
import { defineScene } from 'fumika'
import config from './novel.config'

export default defineScene({ config })(() => [
  { type: 'dialogue', text: '어느새 해가 지고 있다.' },

  // 노을 분위기로 2초(2000ms) 동안 서서히 바꿉니다
  { 
    type: 'mood', 
    mood: 'sunset', 
    intensity: 0.6, 
    duration: 2000 
  },

  { type: 'dialogue', text: '창밖으로 붉은 노을이 길게 드리웁니다.' }
])
```

### 무드 중첩 (Add) 및 공포 연출

```typescript
import { defineScene } from 'fumika'
import config from './novel.config'

export default defineScene({ config })(() => [
  // 위에서 설정한 노을 분위기를 지우지 않고 그 위에 공포 분위기를 "추가(add)" 합니다
  { 
    type: 'mood', 
    action: 'add', 
    mood: 'horror', 
    intensity: 0.8, 
    flicker: 'flicker' // 조명이 고장난 것처럼 깜빡거리게 만듭니다
  },

  { type: 'dialogue', text: '갑자기 전등이 불안하게 흔들립니다.' },

  // 모든 분위기를 완전히 제거하고 원래 화면 상태로 되돌립니다
  { type: 'mood', mood: 'none', duration: 1000 }
])
```

## 주의 사항 (Edge Cases)

| 상황 | 원인 및 해결 방법 |
| :--- | :--- |
| **무드 리셋 방법** | 적용했던 모든 무드 필터를 한 번에 지우고 깨끗한 원래 화면으로 돌아가려면 `mood: 'none'`을 사용하세요. |
| **중첩과 덮어쓰기** | `action: 'add'`를 쓰지 않고 단순히 `mood` 명령어만 호출하면, 기존에 있던 모든 필터가 즉시 지워지고 새 필터로 교체됩니다. 여러 필터를 섞을 때만 `add`를 사용합니다. |
