# 8. 청각적 몰입: 사운드 레이어링

당신은 이제 시각적인 틀을 넘어 청각적인 요소를 설계할 수 있습니다.  
게임의 완성도를 결정하는 핵심은 바로 사운드 레이어링 전략에 있습니다.  
이 페이지에서는 소리로 연출의 밀도를 높이는 기술을 배웁니다.

## 동기

정지된 화면과 적막함은 플레이어의 집중을 방해합니다.  
소리 없는 공포 영화나 박진감 넘치는 액션 씬을 상상해 보십시오.  
아무리 좋은 대사라도 소리 없이 텍스트만으로 전달한다면 그 감동은 절반으로 줄어들기 마련이죠.  
적절한 음악이 없는 장면은 핵심적인 연출 요소를 빠뜨린 것과 같습니다.  

```typescript
// ❌ 밋밋한 연출: 소리 없음
export default defineScene({ config })(() => [
  { type: 'character', name: 'fumika', image: 'idle:normal' },
  { 
    type: 'dialogue', 
    text: '좀 심심하지 않아?' 
  }
])
```

음악을 배경에 깔면 연출의 밀도가 완전히 달라집니다.  
당신이 의도한 분위기에 플레이어가 자연스럽게 몰입할 수 있도록 도와주기 때문이죠.

소리를 활용해 시각적인 빈틈을 채우고 연출의 완성도를 높여 보십시오.

## 기본 사용법

사운드 연출의 첫걸음은 배경음악(`BGM`)과 효과음(`SE`)을 명확히 구분하는 것입니다.  
가장 큰 차이점은 반복 재생 여부와 사용 목적에 있습니다.  
[`audio`](../modules/audio.md) 명령어를 사용해 현장감 있는 사운드 시퀀스를 구성해 보세요.

```typescript
// ✅ 현장감 있는 연출: 사운드 활용
export default defineScene({ config })(() => [
  { 
    type: 'audio', 
    action: 'play', 
    name: 'bgm',      // 'bgm'이라는 이름의 오디오 슬롯(트랙)을 사용합니다
    src: 'bgm-room',  // 실제로 재생할 파일의 키값입니다
    repeat: true,     // 무한 반복 재생합니다
    volume: 0.4 
  },

  { 
    type: 'dialogue', 
    text: '역시 음악이 있으니까 훨씬 좋네요.' 
  },

  { 
    type: 'audio', 
    action: 'play', 
    name: 'se',       // 효과음은 'se' 슬롯을 사용해 배경음과 섞이지 않게 합니다
    src: 'se-heart' 
  },
  
  { 
    type: 'dialogue', 
    text: '호감도가 상승했습니다.' 
  }
])
```

> [!DANGER]
> **`src`** 속성에는 반드시 설정(`audios`)에 등록된 키값을 사용하십시오.  
> 직접 파일 경로를 입력하면 엔진이 소리를 찾지 못해 적막이 흐르게 되니 주의하십시오.  
> **`name`**은 오디오가 재생될 '자리(트랙)'를 의미하며, 같은 이름의 트랙에서 재생하면 기존 소리는 자동으로 교체됩니다.

사운드 밸런스는 작품의 완성도를 좌우하는 핵심 요소입니다.  
배경음이 너무 크면 대사 전달력이 떨어지므로 세밀하게 조절해야 합니다.

> [!TIP]
> 볼륨은 0.3에서 0.5 사이가 적당합니다.  
> 대사와 배경음이 조화를 이루어 자연스럽게 들립니다.

## 점진적 심화

음악이 갑자기 중단되면 플레이어의 몰입이 깨집니다.  
이때 `duration` 옵션(단위: ms)을 사용하면 부드러운 페이드 효과를 줄 수 있습니다.  
이런 작은 디테일 하나가 장면의 분위기를 완전히 바꿔놓기도 하죠.  

```typescript
// ✅ 감성 연출: 페이드 효과
export default defineScene({ config })(() => [
  { 
    type: 'dialogue', 
    text: '이제 작별할 시간입니다.' 
  },

  // 'bgm' 트랙을 3초간 서서히 멈춥니다
  { 
    type: 'audio', 
    action: 'stop', 
    name: 'bgm', 
    duration: 3000 
  },

  { 
    type: 'dialogue', 
    text: '내일 또 놀러 오세요.' 
  }
])
```

섬세한 볼륨 조절 덕분에 청각적 전환이 자연스러워집니다.  
이런 디테일이 모여야 완성도 높은 작품이 탄생합니다.

> [!WARNING]
> 동시 재생은 가급적 3개에서 4개 이내로 제한하십시오.  
> 너무 많은 소리가 겹치면 노이즈가 발생하거나 시스템 부하를 초래합니다.
