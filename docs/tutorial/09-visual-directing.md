# 9. 박진감 넘치는 연출: 카메라와 시각 효과

당신은 이제 정적인 화면을 넘어 역동적인 연출을 구현할 수 있습니다.  
화면을 흔들거나 번쩍이게 하는 것만으로도 장면을 훨씬 효과적으로 연출할 수 있습니다.

시각 효과는 플레이어를 스토리 속으로 이끄는 강력한 수단입니다.  
이제 더 강력하게 만들어 봅시다.

## 동기

정지된 화면은 긴박한 상황이나 충격적인 반전을 온전히 담아내지 못합니다.  
아무리 중요한 순간이라도 화면에 변화가 없다면 연출의 힘이 떨어지기 마련입니다.  
대사만으로는 장면의 긴박감을 전달하기 어렵습니다.

```typescript
// ❌ 정적인 연출: 감정과 화면의 불일치
export default defineScene({ config })(() => [
  { type: 'character', name: 'fumika', image: 'normal:angry' },
  { 
    type: 'dialogue', 
    text: '내 주말이 날아갔다고?!' 
  }
])
```

반면, 대사가 출력될 때 화면이 강하게 흔들린다면 연출의 밀도가 완전히 달라집니다.  
캐릭터의 분노를 시각적으로 즉각 전달할 수 있기 때문이죠.  
이런 시각적 연출은 장면의 몰입도를 크게 높여줍니다.

## 기본 사용법

가장 직관적인 시각 효과는 바로 `shake`입니다.  
폭발이나 타격, 혹은 정신적인 충격을 연출할 때 활용해 보세요. 단 한 줄의 코드만으로 화면에 강렬한 임팩트를 줄 수 있습니다.

```typescript
// ✅ 카메라 흔들림 활용
export default defineScene({ config })(() => [
  { 
    type: 'dialogue', 
    text: '노트북을 부술 듯이 노려봅니다.' 
  },

  // 0.4초간 흔듭니다
  { 
    type: 'camera-effect', 
    preset: 'shake', 
    intensity: 15, 
    duration: 400 
  }
])
```

> [!TIP]
> 사운드 효과와 함께 사용해 보십시오.  
> 화면이 흔들릴 때 효과음이 함께하면 몰입감이 더욱 높아집니다.

## 점진적 심화

`shake` 외에도 색조 변화와 섬광 효과로 더 섬세한 감정을 표현할 수 있습니다.  
분위기를 반전시키거나 특정 사건을 강조할 때 이 기법들을 사용해 보세요.

### 1. 섬광 효과

번개나 정신적 충격, 혹은 깨달음을 얻는 순간에는 `screen-flash`가 효과적입니다.

```typescript
// ✅ 섬광 활용
export default defineScene({ config })(() => [
  { 
    type: 'dialogue', 
    text: '해결책이 번개처럼 당신의 뇌리를 스쳐 지나갔습니다!' 
  },

  // 0.1초간 번쩍입니다
  { type: 'screen-flash', preset: 'white', duration: 100 }
])
```

> [!WARNING]
> 너무 잦은 섬광은 플레이어의 눈을 피로하게 만듭니다.  
> 가장 극적인 순간에만 신중하게 사용하여 효과를 극대화하십시오.

### 2. 무드 필터

회상 장면이나 우울한 감정, 긴장된 분위기를 연출할 때는 `mood` 명령을 사용하세요.  
시각적 분위기를 통해 정보를 텍스트보다 먼저 전달할 수 있습니다.

```typescript
// ✅ 필터 활용
export default defineScene({ config })(() => [
  { 
    type: 'dialogue', 
    text: '주변이 어둡게 가라앉기 시작했습니다.' 
  },

  // 2초간 어둡게 변경합니다
  { 
    type: 'mood', 
    mood: 'night', 
    duration: 2000 
  }
])
```

> [!NOTE]
> 필터를 해제하려면 `mood: 'none'`을 사용하세요.  
> 원래 화면으로 부드럽게 돌아옵니다.
