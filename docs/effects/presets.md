# 🎭 Motion Effect Presets

본 문서는 `camera-effect` 및 `character-effect` 커맨드에서 공용으로 활용되는 엔진 내장 연출 프리셋 명세를 기술합니다.  

---

## 1. 개요 (Overview)

연출 프리셋은 복잡한 수학적 애니메이션 알고리즘을 식별자 하나로 간편하게 실행할 수 있도록 설계되었습니다.  
각 프리셋은 고유한 운동 법칙을 가지고 있으며, `intensity`(강도)와 `duration`(지속 시간) 속성을 조합하여 시나리오의 긴장감과 생동감을 정교하게 조절할 수 있습니다.  

---

## 2. 내장 프리셋 상세 목록 (Presets List)

| 프리셋 식별자 | 기본 지속 시간 | 기본 강도 | 물리적 연출 설명 |
| :--- | :---: | :---: | :--- |
| **`shake`** | `500ms` | `10` | 상하좌우 무작위 진동입니다.  폭발이나 당황스러운 상황에 적합합니다. |
| **`shake-x`** | `500ms` | `15` | 수평(좌우) 방향으로만 날카롭게 흔들리는 효과입니다. |
| **`shake-y`** | `500ms` | `15` | 수직(상하) 방향으로 강한 진동을 부여합니다. |
| **`jolt`** | `200ms` | `30` | 순간적인 강한 충격 후 지수 함수적으로 감쇠합니다.  (타격, 깜짝 놀람) |
| **`bounce`** | `600ms` | `15` | 중력을 가진 물체처럼 위아래로 통통 튀는 사인파 움직임입니다. |
| **`pulse`** | `600ms` | `1.1` | 크기가 주기적으로 팽창 및 수축합니다.  (심장 박동, 강조 연출) |
| **`nod`** | `400ms` | `10` | 고개를 끄덕이듯이 부드럽고 완만한 상하 곡선 운동을 수행합니다. |
| **`fall`** | `800ms` | `15` | 아래 방향으로 가속하며 낙하하는 물리 연출을 수행합니다. |
| **`wave`** | `1000ms` | `20` | 원형 궤적을 그리며 부드럽게 출렁이는 유동적인 움직임입니다. |
| **`float`** | `2000ms` | `10` | 무중력 상태처럼 느리고 우아하게 상하로 부유하는 효과입니다. |
| **`spin`** | `1000ms` | `360` | Z축(화면 정면)을 중심으로 제자리에서 회전합니다. |
| **`spin-x`** | `1000ms` | `360` | X축을 중심으로 수직 회전 연출을 수행합니다. |
| **`spin-y`** | `1000ms` | `360` | Y축을 중심으로 수평 회전 연출을 수행합니다. |
| **`reset`** | - | - | 현재 진행 중인 모든 물리 효과를 즉시 중단하고 원점으로 복구합니다. |

---

## 3. 정밀 조절 가이드 (Advanced Timing)

프리셋의 기본 동작은 커맨드 인자를 통해 시나리오의 요구사항에 맞춰 보정할 수 있습니다.  

*   **`intensity` (연출 강도)**: 효과의 물리적 범위를 결정합니다.  `shake` 등 위치 기반 효과는 픽셀(px) 단위이며, `pulse`는 배율, `spin`은 각도(Degree)를 의미합니다.  
*   **`duration` (지속 시간)**: 한 주기의 완성 또는 전체 효과의 지속 길이를 밀리초(ms) 단위로 설정합니다.  
*   **`repeat` (반복 횟수)**: 효과를 재현할 횟수입니다.  `-1`을 입력하면 명시적으로 `reset` 프리셋을 호출하기 전까지 영구적으로 지속됩니다.  

---

## 4. 연출 활용 팁

*   **강조 인터페이스 구현**: `pulse` 효과를 `repeat: -1`로 설정하여, 플레이어가 반드시 주목해야 할 버튼이나 캐릭터 개체를 효과적으로 강조할 수 있습니다.  
*   **복합 공간 연출**: `camera-effect`를 통해 화면 전체를 흔들면서(`shake`), 동시에 특정 캐릭터에게만 `jolt` 효과를 개별 적용하면 연출의 입체감과 타격감을 극대화할 수 있습니다.  
*   **지속적 환경 조성**: `float`이나 `wave` 프리셋을 긴 `duration`과 함께 무한 반복 설정하여, 몽환적이거나 수중 속과 같은 환경 분위기를 정교하게 조성해 보시기 바랍니다.
