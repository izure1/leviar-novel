# 🎭 Motion Effect Presets

---

## 1. 개요 (Overview)

`camera-effect`와 `character-effect` 커맨드에서 공용으로 사용하는 연출 프리셋 목록입니다. 각 프리셋은 내부적으로 고유한 애니메이션 알고리즘을 가지고 있으며, `intensity`와 `duration`을 통해 세부 연출을 조절할 수 있습니다.

---

## 2. 프리셋 목록 (Presets List)

| 프리셋 | 기본 지속 시간 | 기본 강도 | 설명 |
| :--- | :---: | :---: | :--- |
| `shake` | `500ms` | `10` | 상하좌우로 무작위하게 흔들립니다. (폭발, 진동, 당황) |
| `shake-x` | `500ms` | `15` | 좌우 방향으로만 흔들립니다. |
| `shake-y` | `500ms` | `15` | 상하 방향으로만 흔들립니다. |
| `jolt` | `200ms` | `30` | 지수 함수적으로 감쇠하는 강력한 순간 충격을 줍니다. (타격, 깜짝 놀람) |
| `bounce` | `600ms` | `15` | 위아래로 통통 튀는 사인파 움직임을 보입니다. (기쁨, 강조) |
| `pulse` | `600ms` | `1.1` | 크기가 커졌다 작아지는 효과를 줍니다. (심장 박동, 강조) |
| `nod` | `400ms` | `10` | 고개를 끄덕이듯 부드럽게 위아래로 움직입니다. |
| `fall` | `800ms` | `15` | 아래로 가속하며 떨어지는 연출을 합니다. |
| `wave` | `1000ms` | `20` | 원형으로 회전하듯 부드럽게 출렁입니다. |
| `float` | `2000ms` | `10` | 공중에 떠 있는 것처럼 느리고 부드럽게 상하로 움직입니다. |
| `spin` | `1000ms` | `360` | Z축을 중심으로 제자리에서 회전합니다. |
| `spin-x` | `1000ms` | `360` | X축을 중심으로 회전합니다. |
| `spin-y` | `1000ms` | `360` | Y축을 중심으로 회전합니다. |
| `reset` | - | - | 진행 중인 모든 효과를 즉시 중단하고 원래 상태로 복구합니다. |

---

## 3. 세부 조절 (Advanced Timing)

프리셋의 기본 동작은 커맨드 호출 시 전달하는 인자에 따라 변경됩니다.

*   **intensity (강도)**: 효과의 움직임 범위를 결정합니다. `shake` 등 위치 기반 효과는 픽셀 단위이며, `pulse`는 배율, `spin`은 각도(Degree)를 의미합니다.
*   **duration (지속 시간)**: 한 번의 사이클 또는 전체 효과의 길이를 ms 단위로 설정합니다.
*   **repeat (반복)**: 효과를 몇 번 반복할지 설정합니다. `-1`을 입력하면 명시적으로 `reset`할 때까지 무한 반복됩니다.

---

## 4. 활용 팁

*   **강조 효과**: `pulse` 효과를 `repeat: -1`로 설정하여 중요한 버튼이나 캐릭터를 강조할 수 있습니다.
*   **복합 연출**: `camera-effect`로 전체 화면을 흔들면서(`shake`), 동시에 특정 캐릭터에게 `jolt` 효과를 주어 더욱 강력한 타격감을 표현할 수 있습니다.
